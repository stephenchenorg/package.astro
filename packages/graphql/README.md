## 安裝

```bash
# npm
npm install @stephenchenorg/astro-graphql
# yarn
yarn add @stephenchenorg/astro-graphql
# pnpm
pnpm add @stephenchenorg/astro-graphql
```

## 文件

https://stephenchenorg-astro.netlify.app

## 開發

```bash
pnpm install
pnpm build
```

## 發布套件

```bash
pnpm login
pnpm release
```
