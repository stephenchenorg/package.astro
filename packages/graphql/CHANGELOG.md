## v9.0.0 - 2025-12-19

首次發布。此套件從 `@stephenchenorg/astro` v8.x 拆分出來，包含所有 GraphQL 相關功能。

* 新增模組 `api` - GraphQL API 呼叫相關功能
* 新增模組 `company-setting` - 公司設定的 GraphQL fragments
* 新增模組 `image` - 圖片相關的 GraphQL 查詢與元件
* 新增模組 `page` - 頁面相關的 GraphQL 查詢與元件

請參考 `@stephenchenorg/astro` v9.0.0 的遷移指南以了解如何從舊版本升級。
