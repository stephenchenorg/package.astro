## 安裝

```bash
# npm
npm install @stephenchenorg/astro
# yarn
yarn add @stephenchenorg/astro
```

## 文件

https://stephenchenorg-astro.netlify.app

## 開發

```bash
yarn
yarn build
```

## 發布套件

```bash
npm login
npm run release
```
